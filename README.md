# ThatMagicalOS

ThatMagicalOS is a simple operating system written in Rust (and a little bit of x86_64 assembly)

It's made for the sake of learning how operating systems work, and to have fun doing it.

## How to run?
Make sure you have rust, qemu and grub installed, then just run
```
cargo run --release
```

It should compile the OS, create an ISO and run it in qemu.
